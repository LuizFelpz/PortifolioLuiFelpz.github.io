Projeto de Expanção Airbnb

Situação Problema

A empresa de aluguél de espaços, Airbnb, está passando por um processo de expansão para a cidade de Nova York. Neste cenário, a empresa não detém de muitos conhecimentos sobre essa metrópole e qualquer tomada de decisão errada pode acarretar em uma perda de investimentos enormes. Diante disto, a empresa seleciona uma equipe de Cientistas de Dados com um objetivo único: descrever respostas paras as pergutas do CEO da empresa, sendo o mais fiel possível ao que os dados dizem. Essas perguntas tem o foco de extrair o máximo de informações possíveis para o CEO poder tomar decisões assertivas para o sucesso dessa expansão.

As perguntas do CEO

  • Qual o valor médio do aluguel na cidade de Nova York?
  • Quais os nomes das regiões que existem na cidade de Nova York?
  • Qual o valor do aluguel diário mais caro da cidade de Nova York?
  • Quais são as categorias de imóveis que estão cadastradas dentro da base de dados da cidade de Nova York?
  • Quantos usuários (Hosts) únicos cadastrados existem dentro da base de dados da cidade de Nova York?
  • Como é a variação do preços dos imóveis em NY?
  • Existem mais imóveis baratos ou caros?
  • Qual a distribuição do número de Reviews? Existem imóveis com muitos e outro com poucos reviews?
  • Qual é o valor do aluguel (diária) mais caro de cada região da base de dados da cidade de Nova York, apenas para os imóveis disponível para alugar?
  • Conseguimos saber onde estão localizados os imóveis com o valor do aluguel mais caro, na cidade de Nova York, apenas para os imóveis disponível para alugar?
  • Conseguimos saber onde estão localizados os imóveis pelo seu tipo, apenas para os imóveis disponível para alugar?

Propostas de solução

Depois de uma breve análise das perguntas acima, a equipe de cientistas de dados decide realizar uma apresentação em Power Point para exibir as informações para o CEO e toda a equipe envolvida no projeto de expanção.
Em paralelo, os cientistas também decidem extrair essas informações em Python, pois sua facilidade de manusear dados e automatizar tarefas é bastante alta, e isso interessa os cientistas. Nesse Algoritimo o Python irá extrair os dados de uma planilha e realizar os cálculos necessários, utilizando a estatística descritiva neste caso, e retornará as respostas, como um painel de gráficos, para CEO

Estimação de Data de Entrega Mercado Livre

Situação Problema

A Gigantesca empresa de Marketplace Mercado Livre contém um algorítimo de estimação de tempo de entrega onde o mesmo está tendo "dificuldades" de realizar uma previsão mais acertiva sobre os dados. As reclamações dos clientes sobre a data de entrega são diversas, onde há clientes reclamando que o produto chegou 12 dias depois do previsto e casos onde o produto chegou antes do previsto e, por isto o cliente nao o recebeu pois não estava preparado para receber o mesmo.

Diante deste cenário, a empresa decidiu melhorar este algorítimo de previsão. Para isto ela contratou um time de cientistas de dados para realizar a contrução de um algorítimo que estima o tempo de entrega de um produto de forma mais precisa que o modelo antigo.

Propostas de solução

A Proposta de solução é construir um algorítimo de ML que melhor se adequa os dados fornecidos pela empresa. O time irá contruir um algorítimo que comprove qual modelo é o melhor para este cenário e futuramente os times de desenvolvedores juntamente com os Analistas e Cientistas de Dados irão acoplar este modelo ao site já em produção.

dashboard de tarefas Mei do Mato

Situação Problema

A empresa Mei do Mato, um Instituto de conservação abiental, está iniciando suas operações no mercado de AgroFloresta. Sua instalação está presente em uma fazenda no interior de São Paulo onde a mesma contém tarefas de manejo agroflorestal, manutenção da própria fazenda, manutenção de equipamentos, viveiro de mudas, obras, etc.
Ao progredir em constante crescimento, o corpo diretivo setiu uma dificuldade em gerir todas as tarefas do intituto, pois a mesma não continha nenhuma maneira de controle das terfas do Instituto. Ao perceber que as coisas estavam saindo do controle (tarefas sendo entregues depos do prazo de conclusão, tarefas ficando pendentes por longos períodos, etc), o corpo diretivo decidiu contratar um Analista de Dados para contruir algo que pudesse solucionar este problema.
Em uma pesquisa geral, o CEO do Mei do Mato, decobriu um modelo de tomada de decisão baseada em Dados, o modelo Data Driven que, por sua vez, traz diversos benefícios para ajudar na tomada de decisão. Por este motivo o CEO da empresa contratou um Data Analisys para confeccionar um Dashboard que trouxesse a vista corpo diretivo, todas as informações necessárias para um controle mais efetivo das tarefas do Instituto.

Propostas de solução

Ao analisar o pedido do CEO, o Analista de Dados propôs a contrução de um modelo que alimentasse de forma simples um Dashboardem Power BI. Sua ideia era criar uma planilha que contivesse todas as informações necessárias para a confecção e controle das tarefas do Instituto. Depois da confecção da planilha, o DA iria contruir um modelo de visualização de dados em Power BI e hospedá-lo na nuvem através do Power BI Service.
